<h1>Projeto: Aplicativo Pomodoro Fokus</h1>

<h2>📌 Sobre o Projeto</h2>
<p>Projeto Front-End de um aplicativo de tempo de estudo para ajudar o focu nos estudos, site desenvolvido durante um curso de React-Native.</p>

<h2>🚀 Tecnologias Utilizadas</h2>
<ul>
  <li><b>JavaScript</b> - ReactNative - Expo</li>
</ul>

<h2>✨ Funcionalidades</h2>
<ul>
  <li>Aplicativo para ajudar nas tarefas</li>
  <li>Monitorar o tempo afim de aumentar o focu nos estudos</li>
  <li>Adicionar, editar e deletar tarefas</li>
</ul>

<h2>⚙️ Detalhes Técnicos</h2>
<ul>
  <li>construido com expo</li>
  <li>StyleSheet para estilização</li>
  <li>icones criados com svg e path</li>
</ul>
